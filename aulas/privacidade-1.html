O que é Privacidade Digital?

Privacidade digital é o direito de controlar quais informações pessoais você compartilha online e como elas são utilizadas.

Seus dados são valiosos

Empresas coletam dados sobre você para:

  • Direcionar anúncios personalizados
  • Treinar algoritmos de inteligência artificial
  • Vender para terceiros
  • Criar perfis comportamentais

O que é coletado?

  • Dados de navegação — sites visitados, tempo gasto, cliques
  • Dados de localização — GPS, Wi-Fi, antenas de celular
  • Dados de comunicação — e-mails, mensagens, ligações
  • Dados biométricos — face, voz, impressão digital
  • Metadados — horários, frequências, padrões de uso

LGPD — Lei Geral de Proteção de Dados

Desde 2020, o Brasil conta com a LGPD que garante:

  • Direito de saber quais dados uma empresa possui sobre você
  • Direito de solicitar a exclusão dos seus dados
  • Empresas precisam de consentimento para coletar dados
  • Multas de até 2% do faturamento por violações

Dicas básicas de privacidade

  • Revise as permissões dos apps no celular
  • Use navegador com bloqueador de rastreadores
  • Evite redes Wi-Fi públicas sem VPN
  • Leia (pelo menos resumidamente) as políticas de privacidade
